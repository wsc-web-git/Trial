Summary for Students

Mathematical Logic is the art of thinking clearly with symbols. We use simple statements like p and q to stand for real sentences, and we join them with words like and, or, if...then, and not. A truth table is just a neat chart that shows all possible True and False combinations, so we never get confused.

The main ideas you must remember are: Conjunction (p ∧ q) is true only when both are true. Disjunction (p ∨ q) is true when at least one is true. Implication (p → q) is false only in the case True → False. Biconditional (p ↔ q) is true when both have the same truth value. Negation ( ~p ) simply flips True to False and False to True.

For exams, always write the base columns for p, q, r first and make 4 rows for two variables and 8 rows for three variables. Solve brackets from inside to outside, just like BODMAS in algebra. Keep your eyes on the correct columns, use a ruler if needed, and write each intermediate step. Examiners give step marks, so neat columns matter more than speed.

Finally, check the last column. If it is all True, the pattern is a Tautology. If it is all False, it is a Contradiction. If it has a mix of True and False, it is a Contingency. Practice the standard laws like De Morgan, Distributive, and Conditional laws, because they appear again and again. With calm practice, truth tables become easy scoring questions and build strong logical thinking for mathematics and computer science.
